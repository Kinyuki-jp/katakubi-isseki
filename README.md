# 肩首一席

近畿大学ベースキャンプ周辺向けのInstagram DM予約案内ページです。

## 公開

この `outputs` フォルダの中身をGitHubリポジトリのルートとしてpushしてください。

```bash
git push
```

リモート未設定の場合だけ、初回に `git remote add origin <GitHub repository URL>` を実行してください。

## 予約

サイト上では予約を完結させず、Instagram DMへ誘導します。
利用者には「この時間にマッサージしてほしい」という希望時間を送ってもらう設計です。

- 対応目安: 18:00 - 24:00
- 優先: 夜遅めの時間帯
- Instagram: https://www.instagram.com/kinyukinomati2/
